# Thor

广告位 & 刊例价管理系统

## Getting Started

```
git clone git@git.in.zhihu.com:efe/Thor.git
cd Thor
```

Install dependencies.

```bash
$ yarn
```

Start server.

```bash
$ yarn start
```

If success, app will be open in your default browser automatically.
